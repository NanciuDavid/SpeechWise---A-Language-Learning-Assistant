What is SpeechWise?

SpeechWise is an intelligent assistant that helps users master language pronunciation and enhance overall language proficiency. With tailored exercises, real-time feedback, and interactive features, SpeechWise empowers users to improve their speaking skills and overcome language barriers effortlessly.

How was SpeechWise built?

SpeechWise was built using HTML5, CSS, Bootstrap, and JavaScript. It leverages the Speech API to capture and analyze users' speech in real-time, enabling users to practice pronunciation with instant feedback and interactive lessons.
